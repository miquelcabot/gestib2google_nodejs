# gestib2googleweb

GestIB to Google XML conversion

## Requirements
Requirements: [Node.js](https://nodejs.org/).

To install Express web server and all dependencies, if necessary:
```
$ npm install
```
## Execution
To execute:
```
$ node app.js
```
